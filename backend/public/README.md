# Demo Media Assets

This directory contains demo media files for the demo provider.

## Files

- **broll_city_4s.mp4** - 4 second city b-roll video
- **broll_nature_3s.mp4** - 3 second nature b-roll video
- **hero_image.jpg** - Hero image for demos
- **thumbnail.jpg** - Thumbnail image for demos
- **voice_sample.wav** - Sample voiceover audio

## Note

The placeholder files created here are empty. Replace them with actual media files:
- MP4 files should be short video clips (3-5 seconds)
- JPG files should be sample images
- WAV file should be a short audio sample

These files are served via `/public/*` with 1 hour cache headers.
